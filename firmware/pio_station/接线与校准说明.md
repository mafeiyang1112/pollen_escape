# 接线与校准说明

这份说明按当前 `firmware/pio_station` 的实际默认配置整理，先以“能稳定接通和读数正确”为目标。

## 当前默认接线

### GP2Y1023 三针粉尘模块

| 模块引脚 | 接到开发板 |
|---|---|
| `+5V` | 板上 `5V / VBUS` |
| `GND` | 开发板 `GND` |
| `AO` | `GPIO14` |

注意：

- `AO` 现在默认就是接 `GPIO14`
- 不要再按旧文档接到 `GPIO4`
- 传感器和 ESP32 必须共地

### MAX98357A

| 模块引脚 | 接到开发板 |
|---|---|
| `VIN` | 可先接 `3V3` 试通，再按需要改到 `5V / VBUS` |
| `GND` | 开发板 `GND` |
| `BCLK` | `GPIO5` |
| `LRC/WS` | `GPIO4` |
| `DIN` | `GPIO18` |
| `GAIN` | 先悬空 |

喇叭接在 `MAX98357A` 输出端，不要直接接到 ESP32。

## 简化接线图

```text
ESP32-S3-N16R8                         GP2Y1023
---------------------------------      ----------------
5V / VBUS ---------------------------> +5V
GND ---------------------------------> GND
GPIO14 ------------------------------> AO


ESP32-S3-N16R8                         MAX98357A
---------------------------------      ----------------
3V3 或 5V / VBUS --------------------> VIN
GND ---------------------------------> GND
GPIO5 -------------------------------> BCLK
GPIO4 -------------------------------> LRC / WS
GPIO18 ------------------------------> DIN
```

## 固件默认参数

当前 PlatformIO 固件默认值是：

- `SENSOR_AO_GPIO=14`
- `I2S_BCLK_GPIO=5`
- `I2S_WS_GPIO=4`
- `I2S_DOUT_GPIO=18`

如果你没有改过 `menuconfig` 或 `sdkconfig`，就按上面这组接。

## 上电前检查

1. 传感器 `AO -> GPIO14`
2. 传感器供电接 `5V / VBUS`，不要误接到 `3V3`
3. 传感器和开发板共地
4. `AO` 对地电压不要超过 `3.3V`
5. 如果你还在参考旧图，凡是写着 “`AO -> GPIO4`” 的都以现在这份为准

## 串口里会看到什么

固件会输出类似日志：

```text
sensor raw=xxxmV filtered=xxxmV score=xx.x
```

重点看三个值：

- `raw`
- `filtered`
- `score`

其中：

- `raw_mv` 是 ESP32 在 `GPIO14` 上读到的 ADC 电压
- `filtered_mv` 是平滑后的电压
- `score` 是把电压映射成后端使用的 `pollen_value`

## 推荐校准流程

### 第 1 步：先只接传感器

先不要接喇叭和音频模块，只接：

- `+5V` -> `5V / VBUS`
- `GND` -> `GND`
- `AO` -> `GPIO14`

### 第 2 步：看干净空气下的基线

让传感器稳定 1 到 2 分钟，记录：

- `filtered` 大概稳定在多少 `mV`
- 波动范围有多大

例如稳定在 `620mV` 左右，可以把：

```text
CONFIG_SENSOR_CLEAN_AIR_MV=620
```

### 第 3 步：看高浓度参考值

在安全条件下，让它接触明显更脏的空气，记录高位参考值。

例如稳定在 `1850mV` 左右，可以把：

```text
CONFIG_SENSOR_DENSE_AIR_MV=1850
```

### 第 4 步：确认方向

正常情况应该是：

- 空气更干净，`score` 更低
- 灰尘更大，`score` 更高

如果方向反了，再打开：

```text
CONFIG_SENSOR_OUTPUT_INVERTED=y
```

## 结论

当前这套文档里，传感器接法统一为：

- `AO -> GPIO14`

如果后面还有旧文档残留写成 `GPIO4`，都应该视为过期信息。
